## 工作和开源

对我来说，2025 年最聊不开的话题就是 [Folo](https://folo.is)。这一年来和 DIY、十一、白水等同事们一起工作是我最珍贵的经历。
我有时候觉得自己太菜，经常搞砸事情，很多方面做的也不够好，但是他们对我十分包容和支持。
能把自己最热爱的事情变成工作，这种感觉真的很棒。我爱自由、爱开源、爱 Folo、爱这些朋友们。

![](https://s2.loli.net/2025/12/25/4EeCnXMdc1gItmJ.png)

故事的最后让我感到难过，我总是想如果过去的自己能做得更好更努力一些，或许结果会不一样。
回过头来看，自己今年的表现确实不够好，GitHub 上的绿点也比去年少。

找工作那几天得到了很多朋友的帮助和支持，也收获了很多厉害前辈们的认可，这里也是十分感谢大家。
年底的时候我找到了一份新工作，在 [Dify](https://dify.ai) 做开源，还是远程。
新的领导和同事也很关照我，给了我很多的自由度和支持。
好像都一样，好像哪里也有些不一样。哦对，入职刚一周，就搞了个 bug 🥲。

希望明年能够让自己变得更强一点，多贡献一些自己感兴趣的东西，最好能打磨个自己的产品收获大家的认可。
也希望自己能多挣点钱，可以不用太为生计发愁。

## 旅行和社交

今年最大的一笔支出就是买了辆车，经常让我觉得花了太多不该花的钱。
但现在想象还是觉得不后悔，我很享受在路上在车里的感觉。（甚至在车里也睡过了很多个夜晚）

![](https://s2.loli.net/2025/12/25/HmjCaRpPyzGIfEb.png)

买车之后开车去了很多地方。最长的一段旅程就是开车从安徽去了川西，和同学一起自驾到了雪山边。
车子保险杠第一次刮蹭之后难过的不行，底盘低导致的刮蹭也让我趴着地上看了好久。

![](https://s2.loli.net/2025/12/25/V41BqyJYcSNzG7v.jpg)

找到新工作之后来到了苏州，想着到年前这段线下的时间能多和网友们见见面，交交朋友。
明年也想多出去走走，再扩大一点活动范围，争取可以出次国。
今年还是有太多时间选择窝在老家里了。

## 电子产品和应用

Dify 在 AI 产品上的报销额度十分慷慨。我之前主要用的还是开源免费的 Copilot，觉得什么模型都能用，也挺好的，就是到了月底用量就会不够。
现在是 Claude Code 和 VSCode 混着用。Claude 有一个浏览器插件可以用，这对我这个 Chrome 党很友好，不必用 Atlas。
Claude Code 的话，我不是很习惯在 VSCode 的终端里来使用，就选了 Ghostty 配合着用。
自己一边感叹着 AI 工具的进步和解放的人类生产力，一边又日常被 AI 在乱写气的不行。

Folo 依然是我最喜欢的信息浏览工具，但我也会配合本地的 RSS 阅读器 [NetNewsWire](https://netnewswire.com) 使用。
我把 GitHub 订阅、虎扑篮球新闻这种刷新的比较多比较快的源移动到了本地，在 APP 内来刷新。
放弃了一些反爬变严重的源（比如 b 站），回归到官方的订阅方式。

尝试了离开 Raycast，收窄了我的使用场景。就用默认的 Spotlight 配合 [Supercharge](https://sindresorhus.com/supercharge) 来用。这是我今年最满意的小工具，治好了我对与 macOS 很多的强迫症。macOS 内置的剪切板助手勉强也能用。

今年买了 Mac mini 又卖了，因为自己还是只需要一台便携的笔记本电脑就够了。自己的 m1 pro 也度过了第三个年头。
买了 PS5 并开了会员但是逐渐吃灰，这两个月在苏州也没带过来。
NAS 倒是一直勤勤恳恳的工作着。我很喜欢当个电子仓鼠，把喜欢看的电视、电影、动漫都存在里面。在外面看起来也很方便，手机和车机都能看。

## 情感和家人

今年也是没有对象的一年。日常因为这个事情被家里人念叨甚至是责备。
但我也没什么办法，一个是我也很难遇到合适的人，另一个是我没那么想谈恋爱了，总觉得会很麻烦，自己一个人这么该着也挺好的。
我只有一个独处很久的情况下才会觉得孤独，平时有自己的忙的事情或者和朋友去玩的时候都挺好的。

来年也是希望和身边的人都能好好的，不会有什么矛盾，父母都健健康康的。
